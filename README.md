# 🌡️ Arduino LM35 Temperature Monitor

## 📌 Giới thiệu
Dự án sử dụng cảm biến LM35 để đo nhiệt độ và hiển thị dữ liệu qua Serial Monitor.

## ⚙️ Công nghệ sử dụng
- Arduino
- C/C++
- Proteus (mô phỏng)
- Git/GitHub

## 🔧 Chức năng
- Đọc nhiệt độ từ cảm biến LM35
- Hiển thị nhiệt độ theo thời gian thực
- Có thể mở rộng sang LCD hoặc Web

## 📊 Sơ đồ hoạt động
LM35 → Arduino → Serial Monitor

## ▶️ Cách chạy
1. Kết nối cảm biến LM35 với Arduino
2. Upload code lên Arduino
3. Mở Serial Monitor để xem nhiệt độ

## 📁 Cấu trúc project
- `firmware/`: Code Arduino
- `simulation/`: File Proteus
- `docs/`: Tài liệu
